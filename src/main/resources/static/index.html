Welcome to Yesterdays News API

Category Endpoints

  • Get All Categories:
    Send a GET request to /api/v1/categories.
  • Create a Category:
    Send a POST request to /api/v1/categories with a JSON body: 
    {
    "type": "Category Name"
}
  • Update a Category:
    Send a PUT request to /api/v1/categories/{id} with a JSON body: 
    {
    "type": "Updated Category Name"
}
  • Delete a Category:
    Send a DELETE request to /api/v1/categories/{id}.

User Endpoints

  • Get All Users:
    Send a GET request to /api/v1/users.
  • Create a User:
    Send a POST request to /api/v1/users with a JSON body: 
    {
    "username": "JohnDoe",
    "email": "johndoe@example.com"
}
  • Update a User:
    Send a PUT request to /api/v1/users/{id} with a JSON body: 
    {
    "username": "UpdatedUsername",
    "email": "updatedemail@example.com"
}
  • Delete a User:
    Send a DELETE request to /api/v1/users/{id}.

Article Endpoints

  • Get All Articles:
    Send a GET request to /api/v1/articles.
  • Create an Article:
    Send a POST request to /api/v1/articles with a JSON body: 
    {
    "title": "Article Title",
    "content": "This is the content of the article.",
    "publicationDate": "2025-04-15",
    "userId": 1,
    "categoryIds": [1, 2]
}
  • Update an Article:
    Send a PUT request to /api/v1/articles/{id} with a JSON body: 
    {
    "title": "Updated Title",
    "content": "Updated content of the article.",
    "publicationDate": "2025-04-16",
    "userId": 1,
    "categoryIds": [1, 3]
}
  • Delete an Article:
    Send a DELETE request to /api/v1/articles/{id}.
